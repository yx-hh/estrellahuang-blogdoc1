项目中所遇BUG

Posted on January 5, 2018 本文总阅读量次

1.创建maven project的war工程时遇到pom.xml第一行报错Cannot detect Web Project version. Please specify version of Web Project…如下

在pom.xml的project标签内加入如下代码，指定maven war包的编译版本即可

如果报错依然存在，关闭pom.xml;点击项目名右键，选择maven下的update project..,会跳出一个update Maven project勾选上刚刚建的这个项目，点击ok即可。

2.创建Maven项目后pom.xml报错“web.xml is missing and is set to true”

在src下面创建文件夹WEB-INF然后再创建一个名为web.xml的.xml文件即可。
