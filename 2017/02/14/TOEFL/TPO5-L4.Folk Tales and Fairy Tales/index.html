TPO5-L4.Folk tales and Fairy Tales

Posted on February 14, 2017 本文总阅读量次

Words and Prases

literature [‘lɪt(ə)rə.tʃʊr] 文学；

genre [‘ʒɑnrə] 类型；

start out 开始，出发；

characterize [‘kerəktə.raɪz] 描述；是…的特征；以…为典型；使…具有特点（或最引人注目的特征）

teller n 讲故事的人；

communal [kə’mjun(ə)l] a 共享的，不同群体的；

community 社区；

constant a 不变的；重复的；

thread [θred] 思路，线程；

hold together 团结在一起；

that was the thread that held the tale together.

这是故事团结在一起的思路；意译为，为情节穿针引线；

modify [‘mɑdɪ.faɪ] 修改，调节；

mode [moʊd] 模式，风格；

audience [ˈɔːdiəns] 观众；

improbable [ɪm’prɑbəb(ə)l] adj. 不大可能真实的（或发生的）；不大可能的；奇异的；荒谬的

impossible a 不可能的；

royalty n 皇室；

line-up 节目安排；

Once upon a time 很久以前 (终于不用再说long long ago )

in a faraway [.fɑrə’weɪ] land 在一个遥远的地方；

specify [‘spesɪ.faɪ] 具体说明；

come back to this point in a few minutes

authoritative a 权威的；

be tailored to sb 为…定做；

pass down orally within cultures generation to generation

在某一文化领域内口口相传；

have a slightly different version of the same folk tale

be tailored to suit a particular audience

Sentence

eg: Now we can’t really talk about fairy tales(童话故事) without first talking about folk tales(民间故事) because there’s a strong connection between these two genres, these two types of stories.

fairy 美丽的；优雅的；妖精(一样)的；仙女似的；

eg: a frog [frɒɡ] turning into a prince[prɪns] ;

princess n 公主；

eg: So what’s the folk tale? How would you characterize(描述) them, Jeff?

eg: A number of things happen when an oral tale gets written down.

当口头的故事被写下来的时候会发生大量的改变。

Editor’s message
